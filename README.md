## üí° The Problem
We consume hours of video content (lectures, tutorials, meetings), but our brains struggle to retain linear information. We forget how concepts connect.

## üöÄ The Solution
**VidGraph.ai** is an AI-powered visualization tool that transforms YouTube transcripts into **Interactive Knowledge Graphs**.
Instead of reading a wall of text, you explore a dynamic network of concepts, seeing exactly how ideas relate to one another.

## ‚ú® Key Features
- **AI Concept Extraction:** Uses **Google Gemini Pro** to identify core concepts and relationships.
- **Interactive Visualization:** Drag, zoom, and explore nodes using a physics-based graph engine.
- **Hybrid Input System:** Supports both direct YouTube URL fetching and manual transcript processing for maximum reliability.
- **Dark Mode UI:** A clean, developer-focused interface built with Streamlit.
